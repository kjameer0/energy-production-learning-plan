# Stress and Energy — Module

A single-session schema-building module. The goal is not clinical knowledge or dietary prescription — it is enough structural understanding of the stress-energy system to navigate primary literature independently and evaluate health claims on their own merits.

## Structure

- `goals.md` — mastery targets and metacog checks
- `concept-graph.md` — the DAG: nodes, prerequisites, edges
- `nodes/` — one file per concept node

## How to use

1. Read `concept-graph.md` first — get the terrain before the details (Bear Hunter / first Whole, philosophy §10)
2. Walk nodes in order: 01 → 02 → 03 → 04 → 05 → 06
3. At each node: read the sources, then work through the schema exercises
4. Run metacog checks before advancing
5. After completing the module, use it as the reference for drafting the spaced-review email

## What this module is building

The ability to read a claim like "chronic stress suppresses thyroid conversion and reduces ATP output" and know: (a) what each term refers to anatomically, (b) what the causal chain would need to look like, and (c) what kind of evidence would or wouldn't support it.

This is research literacy grounded in mechanism, not memorization.
